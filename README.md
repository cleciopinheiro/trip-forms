# Formulário /Forms

🇧🇷 PT-BR

Este projeto foi elaborado para treinar HTML, CSS e JavaScript da forma mais pura possível, criando um formulário simples com verificações nos inputs para liberar o envio do mesmo.

<br>

🇺🇸 EN

This project was designed to train HTML, CSS and JavaScript in the purest possible way, creating a simple form with checks on the inputs to release the submission.

<br>
